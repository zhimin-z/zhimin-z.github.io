## Getting Started

To add a new blog post, simply create a new file in the `_posts` directory with the format:

```
YYYY-MM-DD-title-of-your-post.md
```

Each post should have front matter like this:

```yaml
---
layout: post
title: "Your Post Title"
date: YYYY-MM-DD
---
```

Then write your content using Markdown!

## Features

This blog supports:

- **Markdown formatting** for easy writing
- Automatic chronological ordering
- Clean, readable URLs
- Integration with your academic homepage
